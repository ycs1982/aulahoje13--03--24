BANCO DE DADOS NÃO RELACIONAS

Bancos de dados relacionais e não relacionais armazenam

e gerenciam dados de maneiras muito diferentes. As seções a seguir discutem diferenças específicas.

Bancos de dados relacionais armazenam dados em formato tabular e
